# Week 3: Portal Arena 1 vs 1

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

Sound files must be present in `assets/sounds/` before running:

- `punch_1.wav` through `punch_9.wav`
- `win.wav`
- `background.mp3`
- `startmusic.mp3`
- `yoink.mp3`

Image files must be present in `assets/images/` before running:

- `starting.png`
- `fighting.png`
- `ending.png`

**Player 1 Controls:**

- Move: A / D
- Attack: F
- Block: G

**Player 2 Controls:**

- Move: Arrow Keys
- Attack: K
- Block: L

Press **ENTER** to start or rematch.

**Opening the Chrome Console**

- **Windows:** Press `F12` or `Ctrl + Shift + J`, then click the **Console** tab
- **Mac:** Press `Cmd + Option + J`

The console will show any errors in your sketch.
